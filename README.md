## About
Weaponized Autism is a chess engine estimated to around ~2000 ELO.
It uses the Minmax Algorithm with alpha-beta pruning.

## How to use

To install it, run 
```bash
npm i
```
to install all dependencies in package.json.
Now run
```bash
node /src/server/app.js && node .
```
to start the server.
